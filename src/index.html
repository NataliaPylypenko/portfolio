HTML5

CSS3

Bootstrap4

Ця мова стилів дозволяє мені створювати абсолютно будь-який зовнішній вигляд вашого сайту або програми. Все обмежується тільки вашою фантазією!

Java Script

Ця мова програмування дозволяє оживити все що завгодно: слайдери, вікна, підказки, вкладки, отримання даних від сервера і багато іншого

Jquery

Бібліотека Jquery дозволить прискорити розробку. Без необхідності інтегрувати в проект ми її не будемо, але навик роботи з нею присутній

React

Ця бібліотека дозволяє створювати web-додатки. Ми можемо створити максимально інтерактивний продукт саме під ваші цілі

Node.js

Ця платформа дозволяє створювати бекенд для вашого продукту - "мізки", які будуть виконувати дії, які користувач не бачить

Mongo.db
